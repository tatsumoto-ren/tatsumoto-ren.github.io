MDX dictionaries

December 13, 2020 — Tatsumoto Ren

GoldenDict is yet another dictionary lookup program that can be used for learning Japanese. GoldenDict supports MDX dictionaries. It can read EPWING dictionaries as well, but the feature is very clunky in my experience. Qolibri is much better for EPWING. This article provides you with download links for MDX dictionary files.

Downloads

Epistularum made a collection of MDX dictionaries for GoldenDict.

Download links

  • Magnet.
  • Torrent file.

To reduce the size, you can exclude optional directories, such as:

  • 11 他言語 (other languages).
  • 13 書道 (calligraphy).
